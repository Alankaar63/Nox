To use your custom NOX app icon image:
1) In Android Studio: right click app -> New -> Image Asset
2) Foreground Layer -> Image
3) Select your icon file (the NOX image you shared)
4) Name: ic_launcher

Athlete photos for in-app UI:
Place these files in:
app/src/main/assets/athletes/

Required names:
- usain-bolt.jpg
- chris-bumstead.jpg
- michael-phelps.jpg
